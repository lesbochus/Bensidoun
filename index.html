Configuration Supabase
Entrez vos clés Supabase pour connecter l'application à la base de données partagée.
1
Créer un compte gratuit
Allez sur supabase.com → Start for free → créez un projet nommé qr-marches.
2
Créer les tables SQL
Dans votre projet → SQL Editor → collez et exécutez le SQL ci-dessous :
create table users ( id uuid primary key default gen_random_uuid(), nom text not null, email text unique not null, password text not null, role text default 'placier', status text default 'active', created_at timestamptz default now() ); create table merchants ( id text primary key, qr_data text, nom_commercial text not null, responsable text not null, siret text not null, numero_carte text, activite text, telephone text, email text, ville text, marche text, emplacement text, validite text, statut text default 'autorise', created_at timestamptz default now(), created_by text ); -- Désactiver RLS pour usage simple alter table users disable row level security; alter table merchants disable row level security; -- Admin par défaut insert into users (nom,email,password,role) values ('Administrateur','admin@marche.fr','Admin1234','admin');
3
Copier vos clés
Dans votre projet → Settings → API → copiez Project URL et anon public key.

Ces clés sont sauvegardées localement sur cet appareil.
